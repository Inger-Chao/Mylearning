## 时间复杂度与空间复杂度

### 错题

```c
int func(int n){
    int i = 0, sum = 0;
    while(sum < n) 
        sum += ++i;
    return i;
}
/*第 k 次循环 sum = (k*(k+1))/2,故时间复杂度为 O(n ^ 1/2)*/
```

```c
y = 0;
while ((y + 1) * (y + 1) <= n)
    y = y+1;
/* 显然此题是 y^2 = n 时跳出循环，只需要执行根号 n 次，故时间复杂度为 O(n^ 1/2 ) */
```

ps：要注意区分 i = i * 2，此时 2^k = n 故时间复杂度为O(log 2 n)
